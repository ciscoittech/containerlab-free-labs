## 1. Cisco's Strategic AI Pillars

### Pillar 1: AGNTCY - Internet of Agents

Cisco co-founded **AGNTCY** with LangChain and Galileo, donated to Linux Foundation (July 2025).

**Vision:** "The Internet of Agents" - enables any agent to work with any other agent, regardless of builder.

**Core Components:**
| Component | Description |
|-----------|-------------|
| **OASF** (Open Agent Schema Framework) | OCI-based data model for describing agents, supports A2A and **MCP servers** |
| **Agent Directory** | Discovery and announcement of agents across organizations |
| **Agent Connect Protocol** | Standard communication between agents from different frameworks |
| **SLIM** | Secure Low-latency Interactive Messaging for agent-to-agent comms |

**Supported Frameworks:**
- LangGraph
- LlamaIndex
- CrewAI
- **MCP (Anthropic's Model Context Protocol)** - Claude Code uses this!

**Resources:**
- GitHub: https://github.com/agntcy
- Docs: https://docs.agntcy.org/
- Website: https://agntcy.org

Cisco's vision for autonomous IT operations:

> "AgenticOps is a new paradigm for modern IT operations powered by AI agents"

**Products Implementing AgenticOps:**
