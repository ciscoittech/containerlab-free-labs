# Setting Up Damira AI

Damira AI is a free AI assistant for network engineers. It's pre-installed in these Codespaces to help you work through the labs.

## Quick Setup (2 minutes)

### 1. Get Your Free API Key
- Go to [damiraai.com](https://damiraai.com)
- Click **Sign Up** (no credit card required)
- Copy your API key from the dashboard

### 2. Configure the Extension
- Open VS Code Settings (`Ctrl+,` or `Cmd+,`)
- Search for **"Damira"**
- Paste your API key in the **API Key** field

### 3. Start Using Damira
- Open the Damira chat panel (click the Damira icon in the sidebar)
- Ask questions about your lab, paste output, or describe what you're stuck on

## What Can Damira Help With?

- **Troubleshooting**: "My OSPF neighbor is stuck in Init, here's the output..."
- **Concepts**: "What's the difference between OSPF areas and BGP autonomous systems?"
- **Configuration**: "How do I configure eBGP between two FRR routers?"
- **Verification**: "Is this routing table correct for a 3-router OSPF topology?"

## Example Prompts

Each lab's README includes specific Damira prompts you can try. Here are some general ones:
